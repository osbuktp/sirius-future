## P.S.S.
styled - боль, особенно в отсутствии дизайн системы. Приходится плодить сущности исключительно для стилизации отдельных элементов прямо в файле компонента, что превращает его в мусорку (опять же Schedule/Table). Выносить их оттуда в отдельные модули тоже не вариант, так как они в большинстве случаев за пределами этого самого компонента не используются.
